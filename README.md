# QRZ Lookup

WARNING: AI Generated slop follows

Vibe coded slop that lets you query QRZ.com by zipcode. The output file will be a CSV with a list of callsigns in that Zipcode and the number of profile views that callsign has in QRZ. This is a rough proxy for active hams on HF. 

# Usage

```
python3 qrz_lookup.py <zipcode>
```

THe output will be a file called `ham_operators_<ZIPCODE>.csv`
